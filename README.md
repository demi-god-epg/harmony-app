## 📱 About Harmony Lifebond

Harmony Lifebond is a community-based support organization dedicated to building meaningful relationships, financial empowerment, and lifelong mutual assistance among its members.

**Guiding Biblical Principle:**
> "Love your neighbor as yourself." — Leviticus 19:18 • Matthew 22:39 • Mark 12:31

## 🎯 Core Purpose

- **Financial Empowerment**: Provide financial assistance during life's critical moments
- **Community Building**: Create meaningful connections among members
- **Personal Growth**: Support education, entrepreneurship, and skill development
- **Mutual Assistance**: Foster a culture of shared responsibility and compassion

## 📋 Mission & Vision

### Mission
To provide lifelong support that empowers every member to achieve financial stability, personal growth, and an improved quality of life through cooperation, education, business development, and mutual assistance.

### Vision
To become one of Ghana's most trusted and impactful community support organizations, creating opportunities that enable every member to thrive through unity, compassion, and sustainable development.

## 📋 Membership Registration Process

1. Complete Registration Form
2. Upload Required Documents
3. Pay Registration Fee (if applicable)
4. Submit Application
5. Admin Verification
6. Membership Approval
7. Member ID Generation
8. Digital Membership Card Issued

### Required Information
- Full Name & Passport Photo
- Date of Birth, Gender, Nationality
- Ghana Card/ID Number
- Phone & WhatsApp Numbers
- Email & Residential Address
- Occupation & Employer Details
- Marital Status & Emergency Contact
- Next of Kin & Beneficiary Details

## 📥 Getting Started

### Prerequisites
- Node.js 14+ / Flutter SDK
- Git
- Package Manager (npm/yarn/flutter)

### Installation

```bash
# Clone the repository
git clone https://github.com/demi-god-epg/harmony-app.git

# Navigate to project
cd harmony-app

# Install dependencies
npm install  # or yarn install

# Start development server
npm run dev
```

### Build for Production

```bash
# Web PWA
npm run build

# Mobile APK (Android)
npm run build:android

# Mobile IPA (iOS)
npm run build:ios
```
